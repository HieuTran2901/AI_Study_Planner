-----------------------------------

- type

- options

MULTIPLE_CHOICE

- exactly 2 options

- ONLY 1 correct answer

- be concise

- avoid ambiguity
- test understanding rather than memorization

OPTION RULES
-----------------------------------

- optionText
- isCorrect
