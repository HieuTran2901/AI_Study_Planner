- topics must contain ONLY high-level modules (5-7 items)

- difficulty
- order
- estimatedHours
- prerequisites

- resources
- children

- difficulty:
  BEGINNER | INTERMEDIATE | ADVANCED

- order:
  integer only

- description:
  max 120 characters

- imageQuery:
  2-5 word phrase
  no URLs
