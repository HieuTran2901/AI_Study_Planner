# AI Study Planner

A modern, beautiful AI-powered study planning web application that helps students create personalized learning roadmaps, discover courses, and track progress effectively.

![Hero Screenshot](link-ảnh-hero.png)

## ✨ Features

- AI-generated personalized study roadmap
- Smart course discovery
- Progress tracking with analytics
- Study timer with focus mode
- Responsive sidebar + AI Assistant panel
- Smooth animations with Framer Motion

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router v7
